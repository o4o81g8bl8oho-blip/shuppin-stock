# 出品管理 🏷️

メルカリ・PayPayフリマ（Yahoo!フリマ）・ラクマの3サイトに不用品を出品している人のための、**「何がどこに出品中か」を管理するアプリ**。

出品作業そのものは各サイトのアプリで行い、このアプリは「台帳」として使います。

## 主な機能

- 📦 **商品一覧**: 商品ごとに3サイトの状態（未出品/出品中/売れた/取り下げて！）をひと目で確認。出品中は経過日数も表示
- ⚠️ **二重売れ防止**: どこかのサイトで「売れた！」を押すと、他サイトの出品中は自動で「取り下げて！」に変わり、どのサイトの出品を削除すべきか一覧の最上部で警告
- ⏰ **再出品リマインド**: 出品から一定日数（デフォルト14日、変更可）で「そろそろ再出品」バッジ
- 📝 **メモ**: 出品価格や保管場所などを商品ごとにメモできる

## 技術構成

- 単一 `index.html`（vanilla JS）+ `sw.js` + `manifest.json` のPWA
- データはすべて **IndexedDB** にローカル保存。サーバー送信なし・アカウント不要
- オフラインで動作（Service Worker: HTMLはネットワーク優先、その他はキャッシュ優先）

※v1は写真・説明文の下書きストック機能付きでしたが、「物の管理だけでいい」という要望でv2からは状態管理に特化しています。
